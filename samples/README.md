# Sample Output

This folder contains mock output so you can preview what the project is meant to send.

Files:

- `discord_embed_example.json`
  Example Discord webhook embed payload for a successful full run.
- `speed_chart_example.png`
  Dark-mode polished chart image showing the intended `Last 24 Hours` and `Last 7 Days` layout.
- `generate_sample_assets.py`
  Regenerates the mock PNG and sample JSON payload.

These are illustrative samples, not live data from the probe.
